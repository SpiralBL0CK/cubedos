
Documentation Preparation Library
=================================

This folder contains various style and class files that have been used in writing papers for
journals and conference proceedings. They are stored here for reference. Each document that uses
any of these library components has a local copy of that component stored with the document
source. It may make sense to arrange for those documents to use a single copy here instead.

However, since these style and class files are subject to change, it may also make sense for a
*published* paper to have its own archival version of the file(s) that were current at the time
the paper was published. The versions here would be appropriate for papers under development.
